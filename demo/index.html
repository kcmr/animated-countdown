You can apply styles to the <animated-countdown> element in the same manner as you would do with any standard HTML element. The counter text styles are inherited from the <animated-countdown>, but if you need to apply specific styles to the text element (span.counter), you can use the mixin --counter-styles.

.orange {
  font-family: Roboto, sans-serif;
  background-color: #FF892A;
  font-weight: 300;
  color: white;
  font-size: 200px;
  height: 300px;
